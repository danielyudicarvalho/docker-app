## Como rodar

### Instalando dependências
Acesse a pasta `./api` no terminal e execute:
```
npm init
```

e depois 

```
npm install
```

Com isso instalamos as dependências Node que precisamos.

### Levantamento dos contêineres

Acesse a pasta raíz do projeto e execute:

```
docker-compose up -d
```
